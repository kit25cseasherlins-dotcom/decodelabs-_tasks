🍔 Food Ordering Backend API – Task 2 📌 Project Overview

This project is a simple Backend API built using Node.js and Express.js. It demonstrates how to create RESTful APIs to handle basic food ordering operations like viewing food items and managing data using server-side logic.

🚀 Features

Create a simple REST API using Express.js
Get list of food items (GET request)
Add new food items (POST request)
Handle JSON data using Express middleware
Basic data validation and response handling
🛠️ Technologies Used Node.js Express.js JavaScript (ES6)

📂 Project Structure project-folder/ │ ├── server.js # Main server file ├── package.json # Project dependencies └── README.md # Project documentation

▶️ How to Run the Project Install Node.js in your system Open terminal in project folder Install dependencies: npm install Start the server: node server.js Server will run on: http://localhost:3000 📡 API Endpoints ➤ Get all food items GET /foods ➤ Add a new food item POST /foods

Sample JSON Body: { "id": 3, "name": "Sandwich", "price": 80 }
